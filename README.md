# To Do List

A simple web app to manage your daily tasks.

## Features

- Add, edit, and remove tasks
- Save tasks in the browser's local storage
- Sort tasks by start time
- Simple and intuitive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage
